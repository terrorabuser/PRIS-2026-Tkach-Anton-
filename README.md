# Система умных камер видеонаблюдения для фиксации нарушений на дорогах

## Описание проекта

Проект представляет собой интеллектуальную систему видеонаблюдения на основе компьютерного зрения и моделей YOLO (You Only Look Once) для автоматической фиксации нарушений правил дорожного движения и автоматизированного выписывания штрафов.

### Основные возможности:
- Автоматическое обнаружение транспортных средств на видео
- Распознавание нарушений ПДД (превышение скорости, проезд на красный свет, неправильная парковка и др.)
- Фиксация номерных знаков нарушителей
- Генерация отчетов и автоматическое формирование штрафных постановлений
- Веб-интерфейс для мониторинга и управления системой

## Архитектура системы

### Компоненты системы:

1. **Модуль обработки видео (Video Processing Module)**
   - Захват видео с камер видеонаблюдения
   - Предобработка кадров (нормализация, улучшение качества)
   - Извлечение кадров для анализа

2. **Модуль детекции объектов (Object Detection Module)**
   - Использование моделей YOLO для обнаружения:
     - Транспортных средств (автомобили, мотоциклы, грузовики)
     - Пешеходов
     - Дорожных знаков и разметки
     - Светофоров

3. **Модуль распознавания нарушений (Violation Detection Module)**
   - Анализ траекторий движения транспортных средств
   - Определение скорости движения
   - Проверка соблюдения правил проезда перекрестков
   - Обнаружение неправильной парковки
   - Распознавание номерных знаков (OCR)

4. **Модуль базы данных (Database Module)**
   - Хранение информации о нарушениях
   - Хранение видео-доказательств
   - Управление данными о транспортных средствах

5. **Модуль генерации отчетов (Report Generation Module)**
   - Формирование штрафных постановлений
   - Создание статистических отчетов
   - Экспорт данных

6. **Веб-интерфейс (Web Interface)**
   - Streamlit-приложение для визуализации
   - Мониторинг работы системы в реальном времени
   - Просмотр зафиксированных нарушений
   - Управление настройками системы

### Поток данных:

```

    ↓

```

## Установка и запуск

### 1. Создание виртуального окружения

```bash
python -m venv venv
```

### 2. Активация виртуального окружения

**Windows PowerShell:**
```powershell
.\venv\Scripts\Activate.ps1
```

**Windows CMD:**
```cmd
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

**Примечание:** Если в PowerShell возникает ошибка о политике выполнения скриптов, выполните:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск приложения

```bash
streamlit run src/main.py
```

Приложение автоматически откроется в браузере по адресу `http://localhost:8501`
