# Models Directory

## Требуемая модель

Поместите файл модели `license_plate_detector.pt` в эту папку.

### Где получить модель:

1. **Из оригинального репозитория**: Модель доступна на Patreon автора репозитория
   - Репозиторий: https://github.com/computervisioneng/automatic-number-plate-recognition-python-yolov8
   - Модель была обучена на датасете для детекции номерных знаков

2. **Обучить свою модель**: Вы можете обучить свою модель YOLOv8 на собственном датасете номерных знаков

### Структура файлов:

```
models/
└── license_plate_detector.pt  ← Поместите модель сюда
```

### Примечание:

- Модель `yolov8n.pt` загружается автоматически при первом запуске
- Модель `license_plate_detector.pt` необходима для детекции номерных знаков
- Если модель отсутствует, программа продолжит работу, но не будет распознавать номерные знаки
